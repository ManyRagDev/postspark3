# Stack Tecnológica e Padrões

## Backend
* Linguagem: TypeScript (Node.js)
* Framework: Express
* Banco de Dados: PostgreSQL (via Drizzle ORM)

## Frontend
* Framework: React + Vite
* Estilização: TailwindCSS (v4)

## Testes (Obrigatório)
* Comando para rodar testes: `pnpm test` (executa `vitest run`)
* Framework de testes: Vitest

## Padrões de Código
* Commits: Usar Conventional Commits (feat:, fix:, chore:)
* Idioma do código: Inglês
* Idioma da documentação: Português
